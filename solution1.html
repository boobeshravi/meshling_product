Smart street lightings
Block chain enabled street light

Being one of the core technologies of a smart city they involve cloud connected lights with power monitoring and automatic brightness control. We build such smart lights across the city with our innovative block chain technology.

Know More

The challenges

  • Cost

    Building a economical cloud connected street light with reduced AMC charges

  • Robust

    The technology employed needs to be scalable and robust for it seamless operation

  • Technology driven

    The smart light must be driven by ahead of curve technology and must provide required smartness for the entire system

  • Retrofits

    There has been considerable amount of new gen LED lights already been in place and the smart system must retro fit into the exiting street lights

How does it work?

The existing street lights are retrofitted with our smart street light controllers

The smart controllers connect with each other wirelessly establishing a blockchain network

The first street light is connected to a controller hub which would LTE capability to push data to cloud

The block chain network enables the user to have 2 way data transmission to each of the street lights there by retrieving data and controlling them

Monitor and Manage Citywide Lighting With Ease


Our central remote management system allows the user to take complete control of the entire city street lighting infrastructure. Monitor and control each individual street light or group of street lights with an open, secure and intuitive web application suite that is remotely accessible at any time, from anywhere.

Key benefits

Control from any part of the world

The street light would be connected to cloud enabling the user to control from anywhere anytime

Power monitoring

Dedicated hardware withing the controller for precise power monitoring

Scheduling

Schedule switch on/off and set shifts of operation for the lights remotely

Adaptive brightness

Carries required photo sensor to measure environmental conditions and adjust the brightness automatically

Remote error reporting

The device provide instantaneous alerts on problems with the lights and also provides the possibility of limited remote troubleshooting
